# Mentivo

Mentivo is an AI-powered learning platform where people learn web development by actually building real projects, not by blindly copying AI-generated code.

Most AI coding tools help you create things fast, but they do not help you understand what is happening. Mentivo is different. It acts like an AI coding mentor that guides you step by step while you build, teaches concepts in plain English, checks your understanding, and only gives more help when you are genuinely stuck — never just handing you the answer.

When you sign in, you start by describing your project idea in a chat interface. The AI asks questions to understand what you want to build, and if the idea is too large it suggests a focused MVP — a simple single page application or a CRUD application that is actually achievable. It then assesses what you already know, and if you are non-technical it starts from the very basics, explaining things like what a frontend and backend are before writing a single line of code.

From there, the project is broken into small milestones. For each milestone, Mentivo teaches the relevant concept, guides you through writing the code in a browser-based editor, checks your understanding through practical exercises, and moves you to the next step only when you are ready. By the end, you have a finished, working project and a genuine understanding of how web development works.

Mentivo focuses on simple web applications built with HTML, CSS, React, SQLite, and a JavaScript or Python backend. The entire experience runs in the browser with no local setup required.
